##  第一部分总结
- 可变状态至关重要
- 尽量将域声明为final，除非他们是可变的
- 不可变对象（只读对象）一定是线程安全的
- 封装有助于管理复杂性
- 用锁来保护每个可变变量
- 保护同一个不变性条件中的所有变量时，要使用同一个锁
- 执行复合操作期间，要持有锁
